## [Unreleased]

### Changed

- PyPI distribution renamed to **`roomy-observability`** (import package remains **`roomy`**; CLI remains **`roomy`**).

### Added

- Initial monorepo: Python package `roomy`, CLI, SQLite storage, LangChain callbacks.
- Context segments, token estimation, FastAPI read API, Vite + React UI.
- Diagnostics, findings, export, redaction hooks, config, Docker Compose.
